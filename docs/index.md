# Dictator

Документация описывает текущее устройство menubar-приложения после перехода на слои `domain / use_cases / adapters / infrastructure`.

Приложение офлайн-диктовки для macOS на базе MLX Whisper.

Точка входа приложения: парсинг аргументов командной строки,
запуск menu bar приложения и глобальных обработчиков клавиш.

## Что уже доступно

- обзор runtime-слоя и страниц по основным модулям новой структуры;
- автогенерируемая страница [API сборки](api/setup.md);
- ручная архитектурная диаграмма текущих потоков записи, распознавания, вставки и LLM.
